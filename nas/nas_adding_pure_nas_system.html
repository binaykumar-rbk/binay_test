Adding Pure System as a NAS system

To manage and protect data on a NAS share hosted on Pure System, add Pure System as a NAS system to a Rubrik cluster for efficient backup, recovery, and data management.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the NAS Systems inventory card.
    The NAS Systems page appears.
  5. Click Manage NAS Systems.
  6. Click Add NAS System.
    The Add System wizard appears.
  7. In Rubrik Cluster, select the Rubrik cluster that is managing the NAS system.
  8. In System type, select Pure System.
  9. In IP or Hostname, enter the IP address or FQDN of the Pure System hosting the NAS shares.
  10. In API Token, enter the API token.
  11. Optional: In Select automatic discovery protocols, select SMB.
  12. Optional: (SMB credentials) In Username and Password, enter the credentials of an SMB user with backup and restore privileges.
    The password is case-sensitive.
  13. Click Add.

Results

RSC adds Pure System as a NAS system to the Rubrik cluster. You can view the newly added Pure System on the NAS Systems page.
