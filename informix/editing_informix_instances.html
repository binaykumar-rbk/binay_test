Editing Informix instances

Edit the configuration details of an existing Informix instance on RSC, such as updating credentials or changing the IBM Informix user name.

About this task

When editing an instance, you can update the host information or credentials without disrupting the existing data protection policies or SLA Domain assignments.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click IBM Informix.
    The inventory page for IBM Informix appears.
  5. Click Instances.
  6. For the Informix instance to edit, click and select Edit.
    The Edit Informix Instance wizard opens.
  7. Modify the fields.
    FieldDescription
    Instance Name Name of the Informix instance.
    Informix OS Username The operating system username RSC uses to access the host.
  8. Click Save.

Results

RSC saves the configuration changes.
