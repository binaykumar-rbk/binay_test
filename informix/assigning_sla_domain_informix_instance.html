Assigning an SLA Domain to an Informix instance

Assign an SLA Domain to one or more IBM Informix instances to initiate data protection according to the defined policy.

About this task

Assign SLA Domains that include Informix configurations to Informix objects.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Select an instance and click Manage Protection.
    You can select multiple instances.
    The Manage Protection dialog box appears.
  5. Choose the protection option.
    OptionDescription
    SLA Domain Name Protects the selected objects.
    + Creates an SLA Domain.
    Don't protect Excludes the selected objects from further SLA Domain assignments.
    Clear Existing Assignment Assigns selected objects and their contents to the SLA Domain of the next higher level object.
  6. Click Next.
  7. On the Confirmation page, review the details of the assignment and click Save.

Results

RSC applies the selected SLA Domain to the Informix instances, and starts the initial backup job according to the policy.
