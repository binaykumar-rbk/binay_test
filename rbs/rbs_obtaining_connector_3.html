Obtaining Rubrik Backup Service connector

Download the Rubrik Backup Service connector to enable the connected Rubrik clusters to restore folders and files over corrupted data in the file system on the host.

About this task

Skip this task when the Rubrik Backup Service connector is already installed on the connected Rubrik cluster.

The Rubrik Backup Service connector can only be used with the Rubrik cluster from which the software is obtained.

Rubrik provides automatic upgrade of the Rubrik Backup Service connector as part of a general upgrade of the Rubrik cluster software. After upgrading the Rubrik cluster software, the Rubrik cluster automatically upgrades the Rubrik Backup Service for all hosts protected through RSC.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Data Sources and select Connectors.
    The Download Rubrik Backup Service (RBS) page appears.
  4. In Location, select a Rubrik cluster from the drop-down list.
  5. In RBS for the selected location, identify the appropriate connector type and click the download icon.

Results

RSC opens the page to download the Rubrik Backup Service package specific to the selected Rubrik cluster.

What to do next

Install the Rubrik Backup Service software package on the host to provide the Rubrik cluster with the ability to manage data on the host.

For more information about how to install the Rubrik Backup Service software package, see the Rubrik CDM User Guide.
