When Rubrik Backup Service is not installed or disconnected, you can use guest OS credentials to recover virtual machine files.

Installing RBS remotely on multiple hosts

From RSC, you can install RBS remotely to manage and protect multiple Linux, Solaris, HP-UX, AIX, or Windows hosts.

Before you begin

Ensure that the following requirements are met:

  • The Rubrik cluster is connected to RSC.
  • For Linux and other Unix hosts, SSH is enabled.
  • For Windows hosts, winrm https transport is enabled. The Microsoft documentation provides more information about how to configure winrm https transport.

About this task

RSC can automatically deploy RBS to multiple hosts by detecting the operating system of each host and installing the correct software packages.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Data Sources and select one of the following:
    • Linux & Unix
    • Windows
  4. Click Add Hosts.
    The Add Hosts wizard appears.
  5. In the Rubrik Cluster dropdown list, select a cluster from which to deploy RBS.
  6. Under Install RBS on Hosts, select Install remotely.
  7. Click Next.
    The Enter Host Details page opens.
  8. For Host 1, enter the following information.
    FieldDescription
    IP or Hostname Enter an IP address or the name of a host.
    Authentication Method Specify the method used to authenticate the host.
    Host Username Enter the username of the host.
    Host Password Enter the password for the host.
    SSH Host Public Key, for Unix

    or

    HTTPS Thumbprint,

    for Windows    		 Authenticate the identity of the remote host by adding the relevant public key or fingerprint.

    For more information, refer to publicly available documentation on how to obtain the SSH host public key or HTTPS thumbprint.
  9. Optional: To add more hosts, click Add Host.
    Enter the required information for each host.
  10. Click Next.
    The Confirmation page opens.
  11. Verify that the entered information is correct and click ADD.

Results

The Rubrik cluster checks connectivity with the specified hosts and adds the hosts. The cluster also starts the RBS installation on the specified hosts. You can view this information on the Events page.
