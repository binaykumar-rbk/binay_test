Deleting Envoy from the organization network

Delete a connected Rubrik Envoy from the organization network.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Organizations and select Networks.
    The Organization Networks page appears.
  4. Click an organization network.
    The organization network details page appears, displaying the list of all Envoy virtual appliances registered to that organization network.
  5. In the ellipsis menu for the organization network, click Remove Envoy.
    The Remove Envoy dialog box appears.
  6. Click Remove.

Results

RSC removes the Envoy virtual appliance from the organization network.
