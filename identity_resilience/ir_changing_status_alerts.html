Changing status for alerts

Change the status of alerts generated for privilege group membership changes or Group Policy Object modifications.

Procedure

  1. Log in to RSC.
  2. Click and select Identity Resilience.
    The Identity Resilience dashboard appears.
  3. On the navigation bar, click Activity > Alerts.
  4. Select one or more alerts using the checkbox and click Change Status.
    The options to change the status of the selected alerts appear.
  5. Select an option to change the status.
    • Open
    • In Progress
    • Resolved
    • Dismissed
    When you select alerts which have the same status, the specific status will be disabled and you can select from the other statuses available. When you select alerts which have a mixed status, all the status types are available for selection.

    You can filter the alerts to view alerts of a specific status type. When you select the Dismissed status, a confirmation dialog appears to confirm the action. Similarly, when you want to reopen an alert which is already in progress, resolved, or dismissed, a confirmation dialog appears to confirm the action.

Results

You have changed the status for the selected alerts.
