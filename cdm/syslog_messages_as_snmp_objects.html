Syslog messages as SNMP objects

The Management Information Base module inside the Rubrik cluster represents syslog messages as SNMP objects.

The Management Information Base (MIB) module conveys syslog messages by using SNMP. This allows Rubrik clusters that use SNMP to send Rubrik CDM events and provide information for the network management station (NMS) to detect alerts.

The syslog messages conform to the Internet Engineering Task Force (IETF) standards described in the following RFCs:
  • RFC 5425, "The Syslog Protocol"
  • RFC 5676, "The SYSLOG-MSG-MIB"
