Configuring Envoy

Configure the Rubrik Envoy virtual appliance.

Procedure

  1. Log in to the Rubrik Envoy virtual appliance through the vSphere client by using the admin account and the default password: EnvoyAdminPassword
  2. Change the default admin user password by using the following CLI command: system_manage set_admin_user_password
  3. Change the hostname of the Envoy virtual appliance by using the following CLI command: system_manage hostname
  4. Run the following CLI command: envoy_manage envoy_ip_config
    Envoy configures the IP addresses.

Results

You have configured the Rubrik Envoy virtual appliance.

What to do next

Complete the procedures in Registering Envoy on RSC.
