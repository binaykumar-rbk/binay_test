Object filters for snapshot retention

Information about the filters that you can apply to the objects on the Snapshot Retention page. These filters are available on the Data Center and Cloud tabs of the Snapshot Retention page.

Filter Description
Cluster (applicable only for the Data Center tab) The Rubrik clusters connected to RSC. You can select multiple clusters.
Managed by (applicable only for the Cloud tab) Lists the cloud native workload options. You can filter the objects based on the selected cloud native workload.
Supported options are as follows:
  • Azure
  • AWS
  • GCP
Cloud account (applicable only for the Cloud tab) All the cloud accounts associated with the selected cloud native workloads that are onboarded by you. To filter objects, you can select multiple cloud native workloads.
Region (applicable only for the Cloud tab) List of regions associated with a cloud native workload. To filter the objects, you can select multiple regions.
Object type (applicable for the Data Center and Cloud tabs) The object type associated with the connected Rubrik cloud type on RSC. You can select multiple object types.
Object Status (applicable for the Data Center and Cloud tabs) The status of the object. You can select multiple options for the Object Status filter.
Supported values are as follows:
  • Protected (applicable for the Data Center and Cloud tabs)
  • Unprotected (applicable for the Data Center and Cloud tabs)
  • Relic (applicable for the Data Center and Cloud tabs)
  • Remote Unprotected (applicable only for the Data Center tab)
  • Replicated Relic (applicable only for the Data Center tab)
SLA Domain (applicable for the Cloud and Data Center tabs) SLA Domain associated with the object.
Supported values are as follows:
  • Protection SLAs
  • Don't Protect
  • No SLA Domain
Snapshot Type (applicable for the Snapshots page) Type of snapshot.

This filter is available when you click the number in the Snapshot column.

Supported values are as follows:

  • Policy based
  • On demand
Customizations (applicable for the Snapshots page) Filters the snapshot based on custom retention. This filter is available when you click the number in the Snapshot column.
Related concepts
  • Snapshots
    Snapshots and backup information for relic objects and protected objects appear on the Snapshots page.
