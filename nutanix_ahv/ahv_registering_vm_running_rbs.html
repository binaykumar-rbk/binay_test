Registering a virtual machine running the Rubrik Backup Service

After installing Rubrik Backup Service on a virtual machine, register the virtual machine with RSC.

Before you begin

  • Install the RBS software on the virtual machine.
  • Confirm that the virtual machine has ports 12800 and 12801 open.
  • Confirm that the subnet where the Exocompute instance runs has access to ports 12800 and 12801 on the subnet that contains the virtual machine.

About this task

Registering the virtual machine enables RSC to coordinate backup functionality with the Rubrik Backup Service (RBS).

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the AHV VMs inventory card.
    The Nutanix Virtual Machines page appears.
  5. Click the name of a virtual machine on the virtual Machines tab.
    The object details page for the selected virtual machine appears.
  6. Open the ellipsis menu and select Register Rubrik Backup Service.
    A confirmation prompt appears.
  7. Click Register.

Results

RSC launches a temporary Exocompute instance and establishes an authenticated secure connection with RBS running on the virtual machine to register the virtual machine with RSC.
