Taking on-demand snapshots of Dynamics 365 data

Take an on-demand snapshot of Microsoft Dynamics 365 data or metadata.

Procedure

  1. Log in to RSC.
  2. Click and select SaaS App Protection.
    The SaaS Apps dashboard appears.
  3. Click the Dynamics 365 workload.
    The Dynamics 365 dashboard appears.
  4. Click Take on-demand snapshot.
    The Take an On-Demand Snapshot page appears.
  5. Select one of the following types of items for which you want to take a snapshot:
    OptionDescription
    Data Select this option to take a snapshot of individual Microsoft Dynamics 365 table.
    Perform the following steps in sequence:
    • Click Data, and then click Next.
    • Select a table for which you want to take a snapshot.

      You can select multiple tables. You can filter the tables based on the following table types: Standard, Activity, or Elastic. Standard tables store structured data, Activity tables store information about activities or events, and Elastic tables store information that changes frequently and can accommodate various data types and formats.

      Also, you can use the search box to find a specific table.
    Metadata Select this option to take a snapshot of the global metadata, including global option sets used across multiple tables.
  6. Click Take Snapshot.

Results

RSC takes an on-demand snapshot of Microsoft Dynamics 365 data or metadata.
