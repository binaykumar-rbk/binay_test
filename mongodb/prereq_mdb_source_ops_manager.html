Prerequisites for MongoDB sources managed by Ops Manager

RSC has specific prerequisites for protecting MongoDB source clusters managed by the Physical Backup and Recovery via Ops Manager integration method.

A MongoDB Enterprise Advanced license is necessary to have access to Physical Backup and Recovery via Ops Manager integration method.

The following table describes the prerequisites for physical backup and recovery of MongoDB sources.
Prerequisites Description
Port You can enable communication and data transfer by:
  • Opening ports 12800 and 12801 on the MongoDB data hosts and MongoDB Config server.
  • Opening port 9639 on the nodes of the Rubrik cluster.
RBS installation

RSC supports MongoDB Host Deployments for Physical Backup and Recovery via Ops Manager integration on Ubuntu and RHEL Linux distributions. RSC also supports LinuxOne and Linux PowerPC operating systems.

Note:
MongoDB versions 8 and later support Linux PowerPC.
Deployment Configuration with Rubrik

One Rubrik cluster protects a single MongoDB deployment based on Ops Manager. However, one or more Rubrik clusters can protect multiple deployments.
Version and Operating System

MongoDB recommends using Ops Manager version 8.x or later to support all workflows through third-party backup APIs.

The OS Compatibility Matrix lists the supported operating systems based on the configured Ops Manager version.

You can download the release using MongoDB Ops Manager Download.
Host Application Version and Operations Systems

Physical Backup and Recovery via Ops Manager supports replica sets and sharded deployments exclusively on the MongoDB Enterprise Advanced (EA) version of the product.

You can refer to the MongoDB Compatibility Matrix to ensure that your MongoDB deployment is compatible with the Ops Manager version.

You can refer to the Operating Systems Compatible with the MongoDB Agent to ensure that your operating system is compatible with your MongoDB deployment version.
Locale limitation RSC supports MongoDB deployment only in the English language.
