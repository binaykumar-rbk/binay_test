Limitations of MongoDB source clusters managed by Ops Manager

RSC mandates specific limitations on MongoDB source clusters with Physical Backup and Recovery via Ops Manager integration method.

MongoDB source clusters managed using the Physical Backup and Recovery via Ops Manager integration have the following limitations.
Limitation Description
MongoDB upgrade If the MongoDB cluster version is upgraded, you need to trigger a full backup.
Weekly full backups MongoDB recommends taking a weekly full backup for data consistency using the Ops Manager solution. RSC complies with this recommendation by automatically taking a weekly full backup.
