Excluding disks from a GCE instance

Exclude specific disks while protecting other disks attached to a GCE instance. This is useful for managing storage resources efficiently when certain disks do not require backup.

About this task

You can only exclude non-OS disks.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the GCP inventory card.
    The GCP page appears.
  5. Select a GCE instance.
    The GCE instance details page appears.
  6. Click and select Exclude disks.
    The Exclude disks dialog box appears.
  7. Select the disks to exclude.
    Boot disks cannot be excluded.

    Include a previously excluded disk by clearing the excluded disk from the list.

  8. Click Update.

Results

RSC excludes the selected disks from future backups of the GCE instance. However, when an excluded disk is detached from the GCE instance and then reattached, it will be included in subsequent backups.
