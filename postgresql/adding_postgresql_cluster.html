Adding a PostgreSQL cluster

To establish a connection with the Rubrik cluster, add a PostgreSQL cluster by providing the configuration details.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the PostgreSQL inventory card.
    The PostgreSQL page appears.
  5. Click Add PostgreSQL Cluster.
    The Add PostgreSQL Cluster wizard appears.
  6. For Cluster Settings, in PostgreSQL Cluster Name, provide the PostgreSQL cluster name.
  7. In Rubrik Cluster, select a Rubrik cluster from the dropdown menu.
  8. In IP Address or Hostname, select the IP address or hostname of the PostgreSQL cluster.
  9. In PostgreSQL Linux Username, enter the username of the account with access to the PostgreSQL cluster.
  10. Optional: Turn on Advanced Configuration and enter the port number.
    The default port is 5432. If the port is changed, enter the correct port.
  11. Click Add.

Results

RSC initiates a request to add a PostgreSQL cluster to the Rubrik cluster. You can track the status on the Events page.
