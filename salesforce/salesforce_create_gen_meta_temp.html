Creating general metadata seeding templates

Create a general metadata seeding template to specify the general metadata, such as configurations and business processes, to send to the sandbox.

Procedure

  1. Log in to RSC.
  2. Click and select SaaS App Protection.
    The SaaS Apps dashboard appears.
  3. Click the Salesforce inventory card.
    The details page for the Salesforce instance appears.
  4. Click Seed Sandboxes.
  5. Click Create Seeding Template.
    The Create a Seeding Template wizard appears and displays the Template Use Case page.
  6. Select Seed Metadata and then select General metadata.
  7. Click Next.
    The Select Metadata Types page appears.
  8. Select the types of metadata to seed.
  9. For each metadata type, click Select components.
    The Select Metadata Components page appears.
  10. Select the components and click Close.
  11. Click Next.
    The Confirmation page appears.
  12. In Template Name, enter the name of the template.
  13. In Description, enter a description of the template.
  14. Confirm your settings and click Create.

Results

RSC creates the seeding template.
