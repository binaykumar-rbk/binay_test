Adding an Oracle host for CDM version 9.1 or earlier

Add an Oracle host to a Rubrik cluster using CDM version 9.1 or earlier to permit database discovery and protection on that Oracle host.

Before you begin

  • Confirm that all requirements are fulfilled, as described in System requirements for Oracle databases.
  • Confirm that the RBS software is installed on the Oracle host, as described in Installing RBS on Oracle hosts.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the Oracle Databases inventory card.
    The Oracle Databases page appears.
  5. Click Add Hosts.
    The Add Hosts wizard appears.
  6. In Rubrik Cluster, select a Rubrik cluster.
  7. Select I have already installed and registered the RBS software.
  8. Select Add hosts by IP addresses or FQDNs.
  9. In Host IP addresses or FQDNs, enter a comma-separated list of host IP addresses or fully qualified domain names (FQDNs) for the Oracle hosts.
  10. In SYSDBA user, enter the user name.
    The Rubrik cluster supports a single user, which is applicable to all the hosts being registered at this time.
  11. Optional: In Discovery User, enter the name of a Discovery user to use the Discovery user to run queries on the database.
    The Rubrik cluster supports a single Discovery user.
  12. Click Add.

Results

The Rubrik cluster adds the Oracle hosts to a Rubrik cluster, permitting the discovery and protection of the databases on that host.
