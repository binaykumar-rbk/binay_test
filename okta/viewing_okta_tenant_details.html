Viewing Okta tenant details

View the comprehensive details of a specific Okta tenant to check its protection status, review snapshot information, and track recovery and snapshot events in the Events tab.

Procedure

  1. Log in to RSC.
  2. Click and select Identity Resilience.
    The Identity Resilience dashboard appears.
  3. On the navigation bar, click Protection > Inventory.
    The Inventory page appears.
  4. Click the Okta inventory card.
    The Okta page appears with a list of Okta tenants.
  5. In Status, select one of the following:
    StatusDescription
    Active To view only the active Okta tenants.

    This is the default status. These tenants are currently protected and have an assigned SLA Domain.
    Relic To view only the relic Okta tenants.

    These are tenants that have been removed from RSC but whose snapshots are still retained for a defined period. You can only browse and delete the relic snapshots.
  6. Click the tenant whose details you want to view.
    The Overview page appears.

Results

You can view the protection status, assigned SLA Domain, object details, and the calendar view of snapshots for the selected Okta tenant.
