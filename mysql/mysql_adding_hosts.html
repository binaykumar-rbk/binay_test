Adding MySQL hosts

To establish a connection between MySQL and RSC, add a host for performing backup and recovery operations.

About this task

Add a MySQL host along with installing Rubrik Backup Service (RBS) before adding a MySQL instance. RSC provides you the option to install RBS while adding the host.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Data Sources and select Linux & Unix.
    The Linux & Unix page appears.
  4. Click Add Hosts.
    The Add Hosts wizard appears.
  5. In Rubrik cluster, select the Rubrik cluster to connect to the host.
  6. Choose an option for using RBS.
    OptionDescription
    I have already installed and registered the Rubrik Backup service connector. Use the existing installation of RBS.
    I would like to download the Rubrik Backup Service connector now.
    1. In Platform, choose the platform for RBS.
    2. In Version, choose the platform version.
    3. Click .
  7. In Host IP addresses or FQDNs, enter the IP addresses or FQDN of the MySQL host.

    You can specify multiple MySQL host entries here separated by comma.

  8. Click Add.

Results

RSC checks the connectivity with the MySQL hosts and adds the hosts.
