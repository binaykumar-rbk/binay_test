Rubrik Security Cloud workloads

RSC securely manages and protects data from a wide variety of workload types.

Supported workload types:

Cloud workloads

  • Amazon Web Services workloads
  • Google Cloud Platform workloads
  • Microsoft Azure workloads

Container workloads

  • Kubernetes protection

Data center workloads

  • AHV virtual machines
  • File systems
  • Hyper-V virtual machines
  • Microsoft Exchange Server
  • OpenStack virtual machines
  • Oracle Linux Virtualization Manager virtual machines
  • vSphere virtual machines
  • VMware Cloud
  • VMware Cloud Director vApps
  • Windows volume groups

Database workloads

  • IBM Db2
  • IBM Informix
  • Microsoft SQL Server databases
  • MongoDB workloads
  • MySQL
  • Oracle databases
  • PostgreSQL
  • SAP HANA databases

DevOps workloads

  • Azure DevOps
  • GitHub

Identity workloads

  • Active Directory forest
  • Active Directory domains and domain controllers
  • Microsoft Entra ID
  • Okta IdP

SaaS workloads

  • Atlassian Jira
  • Google Workspace
  • Microsoft 365
  • Microsoft 365 Backup Storage
  • Microsoft Dynamics 365
  • Salesforce Data Protection

Storage workloads

  • Managed Volumes and SLA Managed Volumes
  • NAS CD data sets
  • NAS systems
  • SnapMirror Cloud
