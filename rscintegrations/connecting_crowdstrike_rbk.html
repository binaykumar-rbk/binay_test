Connecting CrowdStrike Identity Protection to RSC

Connect CrowdStrike Identity Protection to RSC to get a deeper context of correlated actions for each alert, simplifying your investigation workflow and efficiently remediating identity alerts.

Before you begin

Ensure that you meet the following prerequisites:
  • Creating a CrowdStrike API client and generating a Client ID and Client Secret, as described in Creating CrowdStrike API client.
  • Assigning Alerts Read permission to enable the ingestion of alerts from the CrowdStrike platform to RSC.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Integrations.
    The Integrations page appears.
  4. On the CrowdStrike Identity Protection integration card, click Connect.
    The CrowdStrike Identity Protection page appears.
  5. Click Connect Account.
    The CrowdStrike Identity Protection Integration wizard appears.
  6. Enter the Client ID and Client Secret Key information to enable the integration.
    You can create the CrowdStrike API credentials from the CrowdStrike platform and save the secret key in a secure place.

    If you provide incorrect or invalid credentials, RSC displays an error message to verify your API credentials and permissions before retrying the connection to the CrowdStrike account.

  7. Click Connect.
    RSC performs the following sequence of actions:
    • Verifies the credentials and API client permissions and connects to the CrowdStrike account.

      A confirmation message appears when the integration is successful. You can view the alerts by clicking View Alerts.

    • Ingests the detections from CrowdStrike and displays them on the Alerts page in Rubrik Identity Resilience.

      RSC uses a monitoring capability that syncs every 30 minutes, retrieving all new detections since the last successful synchronization from the connected CrowdStrike account. The last sync details appear on the CrowdStrike integration card.

      Note:
      RSC only displays the new detections and does not display historical detections.

Results

You have successfully connected your CrowdStrike account to RSC.
