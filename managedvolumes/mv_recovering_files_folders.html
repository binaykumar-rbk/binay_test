Recovering files and folders from a snapshot

Use snapshots to search for and recover specific files from Managed Volumes or SLA Managed Volumes.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the Managed Volumes inventory card.
    The Managed Volumes page appears.
  5. Click the relevant tab for the workload type.
    • Managed Volumes
    • SLA Managed Volumes
    The list of workloads appears.
  6. Click an object.
    The object details page for the selected object appears.
  7. Click a date on the calendar to view the list of snapshots taken on that day.
    The day view of snapshots appears.
  8. Open the ellipsis menu for a snapshot and select Recover.
    The Recovery dialog box appears.
  9. Select Recover files and click Next.
  10. Select folders or files within the folders and click Download.

Results

You can find the link to the recovered folders in the Events tab.
