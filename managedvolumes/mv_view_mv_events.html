Viewing Managed Volume events

You can view and monitor Managed Volumes from the Events tab.

Procedure

  1. Log in to RSC.
  2. On the navigation bar, click Events.
    The Events page appears.
  3. Filter the page by Object Type or Event Type.

Results

The page displays the status of tasks for the Managed Volume or SLA Managed Volume.
