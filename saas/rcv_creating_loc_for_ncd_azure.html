Creating RCV locations for NAS CD objects on Azure

Create a Rubrik Cloud Vault location on Azure for assigning to NAS CD objects.

Before you begin

Review the best practices, as described in Rubrik Cloud Vault best practices.

About this task

While creating a Rubrik Cloud Vault (RCV) location for NAS CD objects, you cannot edit the following configuration values:
  • Encryption Type is Server Side
  • Immutability is Always On

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Infrastructure > Rubrik Cloud Vault.
    The Rubrik Cloud Vault dashboard appears.
  4. Click RCV Locations.
    A list of RCV locations appears.
  5. Click Create RCV Location.
    The Create an RCV Location wizard appears, displaying the supported workload types.
  6. Optional: Click the ​allowlist the RCV locations​​ link.
    The list of endpoints that you need to allowlist appears. You can either copy the individual endpoints or download the list of endpoints in a text file.
  7. Select NAS Cloud Direct and click Next.
  8. Select the NAS CD clusters that you want to protect using RCV as the archival location and click Next.
    The Location Details page appears.
  9. In Location Name, enter the Rubrik Cloud Vault location name.
  10. In Cloud Provider, select Azure.
  11. In Tier, select a storage tier.
    TierDescription
    Backup The Backup tier in RCV requires a minimum retention period of 30 days. You can choose this option for disaster recovery use cases where you need fast access to the data.
    Archive The Archive tier in RCV requires a minimum retention period of 180 days to access data. You can choose this for long-term data retention that does not require instant access. The recovery process can take several hours up to days to complete.
    The Rubrik Cloud Vault storage tiers topic provides more information and a comparison between the storage tiers to help you make an informed decision based on your data storage requirements.
  12. In Region, select the region and click Next.
    The Confirm Rubrik Cloud Vault details page appears.
  13. Review the details and click Create.

Results

RSC creates the RCV location.

It may take up to 15 minutes to create the RCV location and synchronize the location with the NAS CD cluster. After synchronization is complete, the RCV location is available for you to create protection policies in the NAS CD web UI.
