Adding a Zscaler integration in RSC

To secure sensitive data in your production environment by combining Rubrik Data Security Posture and Zscaler Data Loss Prevention services, add a Zscaler integration in RSC using the Integrations page.

Before you begin

  • Ensure that Rubrik Backup Service (RBS) is installed on the drop target virtual machine.
  • Ensure that the RSC user has Zscaler DLP system configuration permissions.

About this task

This topic describes a feature or features that Rubrik is gradually rolling out to customers.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Integrations.
    The Integrations page appears.
  4. Click the Zscaler integration card.
    The Zscaler page appears.
  5. Click Add Target.
    The Add Zscaler Integration wizard appears and displays the Select Service Account page.
  6. Select a service account to use for accessing the data.
    Alternatively, you can click the plus icon to add a new service account.

    The service account automates finding sensitive data and exporting it to the designated drop target virtual machine.

  7. Click Next.
  8. In Integration Label, enter a name for the drop target virtual machine.
  9. In Workload, select the workload that contains the sensitive data you want to protect.
  10. In Drop Target, select a virtual machine from your inventory as the drop target.
    Rubrik recommends you use a dedicated virtual machine as the drop target.
    Note:
    The operating system of the workload virtual machine that contains the sensitive data you want to protect and the operating system of the drop target virtual machine must be the same.
  11. In Drop Target Path, enter a valid path on the drop target virtual machine to which you want to export the sensitive data.
    The drop target virtual machine and path you use must be secure, to prevent unauthorized access and to ensure the exported sensitive data remains safe.
  12. Click Next.
  13. Select a method to assign policies to the workload virtual machines.
    OptionDescription
    Assign all sensitive data policies to the target Discovers sensitive data according to all available data policies.
    Assign specific sensitive data policies to the target Provides an option to select specific policies you want to use for discovering sensitive data.
  14. Click Add.

Results

You have added the Zscaler integration in RSC.
