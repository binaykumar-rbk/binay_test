Viewing activity details

You can view activities about a specific task running on RSC.

Procedure

  1. Log in to RSC.
  2. Open the app tray and select Data Protection.
    The Data Protection dashboard appears.
  3. In Dashboards, select Task Monitoring.
    The Task Monitoring dashboard appears.
  4. Select the tab that reflects the status of your task.
    • In Progress
    • Failed
    • Completed
    • Scheduled
    • Cancelled
  5. In Activity Details, click View for the object for which you want to review the task activity details.
    The Activity Details page appears for the selected object.
  6. Optional: Click Download Server Logs to save the server log externally.
    RSC adds the server log to the download queue.

Results

The details for the selected task appear.
