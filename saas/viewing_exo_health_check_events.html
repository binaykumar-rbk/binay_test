Viewing Exocompute events

View the Events page from the Dashboard to see the Exocompute events.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Events.
    The Events page appears.
  4. Click Event Logs.
    This page displays the Event Timeline, cluster, Object Type, Object, Event Type, Location, Duration, Start Timestamp, and Organization details associated with the event.
    The Events Logs page appears.
  5. Optional: Select filters to apply to the events.

Results

RSC fills the Event Logs page with the aggregate values based on the selected filter and displays them along with the Exocompute events. You can click an event to view the series of events leading up to it.
