Recovering files to Windows with RBS

Recover files at the volumes level or at the level of individual files for a volume group on a Windows host with RBS.

Procedure

  1. Log in to RSC.
  2. Open the app tray and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the Windows Volumes inventory card.
    The Windows Volumes page opens.
  5. Click a host name.
    The Windows Volume group details page opens.
  6. In the Snapshots calendar, select a date with a snapshot.
    The list of snapshots for that date appears.
  7. Click Recover in the row of the volume group to recover.
    The Volumes Recovery wizard opens.
  8. Select Recover files and click Next.
    The Volumes Recovery wizard opens.
  9. Select the files and/or folders to recover and click Next.
  10. Select a recovery option.
    OptionDescription
    Download Creates a link to download the selected file or folder.
    Overwrite original Restores the selected files and folders to the original path, overwriting the existing file or folder.
    Recover to separate folder Restores the selected items to the selected location.

    This option requires inputting the full path to the restore location in Folder Path.
    Export to a different host Recovers the selected items to a different host.

    This option requires selecting the export host from the dropdown options.
  11. Click Recover.
    A notification on the page confirms the recovery. This is also indicated on the Events page.

Results

The Rubrik cluster recovers the files based on the selected option.
