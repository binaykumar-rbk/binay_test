The Snapshots card calendar view for filesets

The calendar view of the Snapshots card displays information at different levels of granularity.

The Year and Month views in the calendar display information about backups created for an entire year and month, respectively. A color spot indicator on a specific date indicates backup activity and displays the SLA Domain compliance status for that day.

The Day view displays the details of the individual snapshots that were created on the selected day. The Day view is also a starting point for the recovery actions available for each fileset snapshot.

The following table describes the details available in the Day view for a fileset snapshot. When static retention is enabled, each snapshot entry expands to display the location type and the expiration date of the snapshot.
Detail Description
Time The time when the snapshot was created.
Fileset Name The name of the fileset corresponding to the snapshot.
Files The total number of files in the snapshot.
Location The Rubrik cluster associated with the fileset.
Location type Cluster The Rubrik cluster where the snapshot is stored.

Location type takes one of these values: Local, Archival, or Replication.
Expiration Date The date and time when the snapshot will expire.
