Data Security Posture for NAS CD data sets

For NAS CD data sets that have a Rubrik Cloud Vault location configured, you can enable Data Security Posture to find data sets matching a specified data policy within snapshots.

This topic describes a feature or features that Rubrik is gradually rolling out to customers.

Data Security Posture identifies sensitive data matching a specified policy within archived backups. You can enable Data Security Posture for NAS CD data sets that have a Rubrik Cloud Vault archival location configured.

To enable Data Security Posture for NAS CD data sets, you must assign policies in Data Security Posture. You can also create custom policies and analyzers according to your requirements and assign them to the objects. After the policy is enabled, Data Security Posture performs a full analysis, and the sensitive data classification becomes available in the object details page in Data Security Posture.
