Removing Azure Private Container Registry configurations

When you no longer want to use your Private Container Registry and you want Exocompute to pull images using a registry Rubrik hosts, remove the Private Container Registry configuration.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Cloud Accounts and select Azure.
    The Azure cloud accounts page appears.
  4. From the Azure dropdown menu, select Rubrik Compute.
    A list of Azure Active Directory (AD) domains appears.
  5. Select an Azure AD domain.
    The list of Exocompute host subscriptions appears.
  6. Click for the corresponding Exocompute host account and select Edit Exocompute.
    The Edit Exocompute wizard appears.
  7. Turn off Private Container Registry.
  8. Click Next.
  9. In the subsequent page, edit the regions, subnet, and VPC details as required.
  10. Click Submit.

Results

RSC removes the Private Container Registry configuration.
