Recovering all Teams standard channel content to the original channel

Use indexed Microsoft 365 snapshots to recover all content from a Microsoft Teams standard channel to the original channel.

Before you begin

Ensure that the account is authorized to perform a full Teams channel recovery. Authorization is a one-time process managed by the global administrator.

About this task

For a full channel recovery, all files and messages are recovered for the Teams channel.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
  3. On the navigation bar, click Inventory.
  4. Click the Microsoft 365 inventory card.
  5. From the Selected Subscription dropdown menu, select the subscription.
  6. Click Enter Subscription.
  7. Click Teams and select a team.
  8. On the details page for the team, select a snapshot from the calendar view and click Recover.
    Alternatively, you can search for an object to recover.
  9. Click Recover selected content for one channel type and click Next.
  10. all
  11. Click All channel content > Standard channels and then click Next.
  12. Select the channel to recover and then click Next.
  13. Click Recover to original channel and then click Next.
  14. Select a recovery option.
    OptionDescription
    In-place recovery Recovers files to their original location in the selected directory. In-place recovery recovers only objects in folders/files/attachments.

    In-place recovery of conversations is not supported.
    Recover to a new folder Recovers files to a new folder in the target location.
  15. For in-place recovery, select a recovery option.
    OptionDescription
    Append duplicate objects Creates duplicates of the objects and retains the original objects. RSCcreates the duplicates at the file level, not at the folder level.
    Overwrite duplicate objects With duplicate objects, RSC overwrites the old objects with the objects to be recovered. RSC overwrites files at the object level, not at the folder level.
  16. Click Next.
  17. Log in to the Microsoft 365 account to provideRSC permissions to recover objects to the selected channel and then click Next.
  18. Review the information and then click Recover.

Results

RSC schedules a task to recover the selected Teams channel to the original channel. The restored messages appear with the original author name and original message time attached to the beginning of each message. Files appear on the File tab, retaining their original file names.

For in-place recovery of conversations with attached files, RSC recovers the attached files to the same path under a folder named Source_Channel_Name, where Source_Channel_Name is the name of the source channel to which the attached file originally belonged.

You can track progress for this activity on the Events page.
