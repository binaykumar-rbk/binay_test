Configuring single sign-on in RSC

Configure SSO in RSC by uploading the metadata file of the identity provider and downloading the RSC metadata file.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Users &Access and select SSO Configuration.
    The SSO Identity Providers page appears.
  4. Click Add New SSO IDP.
    The Add RSC Details to Your SSO Identity Provider wizard appears.
  5. Click Next.
    The Add SSO IdP Details to Establish Connection dialog box of the wizard appears.
  6. Upload the ADFS metadata file by dragging and dropping the file into the drag and drop box of the Add SSO IdP Details to Establish Connection dialog box.
    RSC populates the Identity Provider Entity ID, Single Sign-on URL, and Identity Provider Signing Certificate fields with values extracted from the metadata file.
  7. Click Add.

Results

RSC configures SSO for ADFS.

What to do next

Test the RSC SSO connection, as described in Testing the SSO configuration.
