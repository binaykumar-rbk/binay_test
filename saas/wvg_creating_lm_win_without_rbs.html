        • Rubrik Backup Service

          The Rubrik Backup Service is software installed on a host that coordinates activity between the Rubrik cluster and the host.

        • Adding a Windows host

          Add a new Windows host to the Rubrik cluster to begin managing and protecting it.

        • Protecting a Windows Volume group

          Protecting Windows volumes uses the Rubrik Backup Service on a Windows host to read data from the protected volumes, and in some cases to create a Virtual Hard Drive file.

          The Rubrik Volume Filter Driver improves the performance of incremental backups of protected volumes.

          In addition to policy-based snapshots, use the on-demand process to take snapshots of a Windows Volume group.

Creating a Live Mount on Windows without RBS

Create a Live Mount of a volume group to a Windows host without RBS installed through the OS functionality.

Procedure

  1. Log in to RSC.
  2. Open the app tray and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click the Windows Volumes inventory card.
    The Windows Volumes page opens.
  5. Click a host name.
    The Windows Volume group details page opens.
  6. In the Snapshots calendar, select a date with a snapshot.
    The list of snapshots for that date appears.
  7. Open for the snapshot you want to restore and select Recover.
    The Volumes Recovery wizard appears.
  8. Select Live Mount and click Next.
    The Volumes Recovery Wizard opens on the Select mount points page.
  9. Select the mount points to recover for the selected snapshot and click Next.
    The Select host page appears.
  10. Select No Host and click Next.
    The SMB configuration page appears.
  11. On the SMB configuration page, input the required details based on the cluster SMB security configuration.
    OptionDescription
    SMB security is disabled Requires the following fields:
    • Cluster

      The system automatically populates this field.

    • Client IPs

      Input a list of IP addresses for the hosts that require access to the SMB share.
    SMB security is enabled Requires the following fields:
    • Cluster

      The system automatically populates this field.

    • Domain
    • Username
    • Active Directory Groups
    • Client IPs
  12. Click Finish.

Results

The Rubrik cluster mounts the selected volumes.
