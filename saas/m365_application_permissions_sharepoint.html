    • Microsoft 365

      Using the data management and protection services offered by Rubrik Security Cloud, you can protect Microsoft 365 data operations.

      • Microsoft 365 dashboard

        The Microsoft 365 dashboard provides important insights into your protected Microsoft 365 environment. By using the dashboard, you can view and monitor information such as deployment locations and settings, backup and recovery status, events, and compliance information.

      • Microsoft 365 immutability

        RSC uses immutable storage for Azure Blob Storage to provide immutability protection for Microsoft 365 when creating an Azure storage account for Microsoft 365 deployments. Immutability protection ensures that your data remains unaltered, which is critical for disaster recovery, protection from ransomware attacks, and data compliance.

      • Implicit permissions and Microsoft 365 objects

        Rubrik Security Cloud uses implicit permissions to help provide access to specific objects in the Microsoft 365 hierarchy.

      • Protection hierarchy for Microsoft 365 resources

        RSC provides hierarchy-based, automatic protection of Microsoft 365 resources.

        Lists the firewall rules needed for communication between the RSC, Microsoft 365, and Microsoft Azure when using customer-hosted deployment.

      • Microsoft 365 protection

        Use SLA Domains to apply the data protection policies and management policies to Microsoft 365 resources.

      • Backup and recovery with Microsoft 365 snapshots

        RSC protects and manages Microsoft 365 data using snapshots of source Microsoft 365 objects. You can take snapshots of backup data for Microsoft Exchange, OneDrive, SharePoint, and Teams to recover lost or compromised information.

      • Relic Microsoft 365 management

        Microsoft 365 does not support user accounts once the user license is revoked. RSC archives inactive user accounts without snapshots and manages the ones with snapshots as relics.

      • Groups in M365

        RSC uses groups to assign data protection and management policies uniformly to Microsoft 365 objects.

      • Data Observability for Microsoft Sharepoint and OneDrive

        To provide Data Observability, you can enable Anomaly Detection and Threat Monitoring to detect ransomware attacks and configure Data Security Posture to identify sensitive data in backups of Microsoft OneDrive and Sharepoint objects.

    • Microsoft 365 Backup Storage

      With the Microsoft 365 Backup Storage feature, you can back up Microsoft 365 applications such as SharePoint sites, OneDrive user accounts, and Exchange mailboxes. You can also restore large volumes of data during ransomware attacks, accidental deletion, or malicious data deletion and overwrite events.

SharePoint Azure Enterprise applications permission settings

Permissions settings for RSC operations using the Microsoft 365 API and SharePoint.

The following table describes the permissions that the Azure Enterprise application used by RSC requires to protect SharePoint.
Requirement Value Description
Type Application RSC accesses SharePoint APIs on behalf of the Azure Enterprise application, or Application, not the user.
Status Granted for tenant_name Set for each permission, where tenant_name is the name of the affected tenant. Granting tenant-wide admin consent to an Azure Enterprise application allows all users to access the Azure Enterprise application unless otherwise restricted.
Required admin consent Yes Set for each permission.

In addition, the Azure Enterprise application used by RSC must have the SharePoint API permissions described in the following table.

Microsoft API Permission name Description
Microsoft Graph Sites.Read.All Reads items in all site collections
Microsoft Graph Files.ReadWrite.All Reads and writes files in all site collections
Microsoft Graph User.Read.All Reads all users' full profiles
Microsoft Graph Sites.FullControl.All Has full control of all site collections
Office 365 SharePoint Online Sites.FullControl.All Has full control of all site collections
