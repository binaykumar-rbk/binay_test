Resetting passkeys for a user

When a user is locked out and is unable to authenticate access using any of the preconfigured passkeys or TOTP, the administrator can reset passkeys to enable user authentication.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Users & Access and select Users.
  4. Select a user.
  5. Click above the list of users and select Reset Passkeys.
    You can also select multiple users.
  6. In the dialog box that appears, click Reset.

Results

RSC removes all the configured passkeys for the selected users.

What to do next

Users can create new passkeys after their passkeys are reset.
