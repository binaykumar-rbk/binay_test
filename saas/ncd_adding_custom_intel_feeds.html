Adding custom intel feeds for NAS CD

Create a custom intel feed to monitor your NAS CD data sets for specific Indicators of Compromise (IOCs) tailored to the security needs of your organization. While Rubrik provides built-in, vetted threat intelligence, custom feeds enable you to target unique or emerging threats immediately.

Before you begin

Review the following key requirements and limitations to ensure your custom intel feed is compatible:
  • Feed capacity: You can create up to seven custom intel feeds, with each feed containing up to 10 Indicators of Compromise (IOCs).
  • Supported IOC type: NAS CD supports IOCs for a custom intel feed with only file hashes.
  • Unsupported IOCs: NAS CD does not support IOCs with YARA rules for Threat Monitoring.
  • Immediate scanning: Adding a new file hash or enabling a Threat Monitoring for your NAS CD data sets initiates an immediate full scan of your data sets to check for file hashes.

About this task

To monitor data for specific threats, you can create a custom intel feed of IOCs. The Rubrik Threat Monitoring feature proactively scans for threats based on vetted threat intelligence from multiple sources and also enables you to scan your data using custom intel feeds that you define.

Procedure

  1. Log in to RSC.
  2. Click and select Data Threat Analytics.
    The Data Threat Analytics dashboard appears.
  3. On the navigation bar, click Intel Feed.
    The Intel Feed page appears.
  4. Click Add Intel Source.
    The Add a Threat Intel Source wizard appears.
  5. Click Add IOC.
    The Add IOC dialog box appears.
  6. In the IOC Type dropdown list, select File hash.
    NAS CD does not support Threat Monitoring using IOCs with YARA rules.
  7. Optional: In Threat Family, enter the threat family name for the IOC.
  8. In the dialog box, enter the comma-separated file hashes as text.
  9. Click Add.
    You can add multiple IOCs.
  10. Click Next.
    The Source Details page appears.
  11. In Intel Source Name and Description, enter a name and an optional description for this intel source.
  12. Click Add.

Results

RSC adds the custom intel feed and uses this feed to monitor threat when you enable Threat Monitoring for a NAS CD cluster.

You can edit or remove a custom intel feed, or add more IOCs to this feed. For more information, see Custom intel feed.
