Overview tab actions list

The Overview tab on the local page of a fileset host and failover cluster service provides access to actions related to fileset management and protection.

Action Description
Ellipsis menu > Edit object
  • Opens a dialog box where you can modify the IP address or the hostname associated with the object when the object is a host.
  • Opens a dialog box where you can modify the service name and the virtual IP address of the object when theobject is a service.
Ellipsis menu > Delete object
  • Opens a confirmation dialog box with a warning that describes the consequences of deleting hosts when theobject is a host.
  • Opens a confirmation dialog box with a warning that describes the consequences of deleting services when the object is a service.
(Fileset host) Ellipsis menu > Disable Data Threat Analytics events Opens a dialog box where you can disable the email alerts for all Data Threat Analytics events.
Manage Protection Opens a wizard that enables:
  1. Association of filesets with the hosts or services.
  2. Assignment of an SLA Domain to manage the fileset protection.
Take On Demand Snapshot

Opens a wizard to initiate an on-demand snapshot for a fileset on a host or a service.

Backup Window settings defined for an SLA Domain do not apply to on-demand snapshots. Only the maximum retention and remote configuration settings of the associated SLA Domain apply to on-demand snapshots.
Related tasks
  • Removing a fileset host
    You can delete the host from the Rubrik cluster to stop protecting the fileset data on a host.
  • Protecting a host fileset
    Create a host fileset by pairing a fileset with a host. Assign an SLA Domain to the host fileset to protect the data in the host fileset.
  • Taking on-demand snapshot of a host fileset
    Create a backup by taking an on-demand snapshot of a host fileset.
