Fileset host management actions

For every fileset host, RSC provides the ability to check the connection status, to update the certificate, and to edit the identity of the host.

Host-level management actions are accessible from the ellipsis menu available for each host.

Management action Description
Check connectivity Checks the following:
  • Whether the Rubrik Backup Service (RBS) on the host is reachable.
  • Whether the host is reachable through ICMP or echo port.
Update certificate Checks if the RBS certificate on the host is valid or not.

Updates the certificate if expired or close to expiration.
Edit host Allows edits only to the IP address or the hostname.

Does not allow any edits to the Rubrik cluster. To connect the host to a different Rubrik cluster, the host must be removed from the current Rubrik cluster and added to the new Rubrik cluster.
