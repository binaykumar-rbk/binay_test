Upgrading Rubrik CDM from RSC

After you have downloaded the Rubrik CDM software upgrade package and ensured that no precheck errors exist, upgrade the Rubrik CDM version of a Rubrik cluster from RSC.

Before you begin

Follow the instructions in Downloading an upgrade package to download the upgrade package for the selected Rubrik CDM version.

About this task

Note:
You must use a fast upgrade operation to roll back a Rubrik cluster. You cannot roll back a failed Rubrik cluster using the rolling upgrade operation.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Clusters and select CDM Software Upgrades.
    The Rubrik CDM Upgrades page appears.
  4. Select a Rubrik cluster with the upgrade status Ready for Upgrade and click Upgrade.
    You select and upgrade multiple Rubrik clusters from the same starting version to the same finishing version.
    The Upgrades wizard appears.
  5. In the Upgrade Type column, select the upgrade type.
    Upgrade typeDescription
    Fast Performs upgrade simultaneously on all nodes in the Rubrik cluster.

    Fast upgrades may require some downtime for the Rubrik cluster.
    Rolling Performs upgrade serially on the nodes in the Rubrik cluster.

    Only one node is upgraded at a time. Other nodes remain active, ensuring zero downtime.

    The rolling upgrade feature is available only for Rubrik clusters running Rubrik CDM version 8.1.2 or later.
  6. Perform the tasks according to the number of clusters you are upgrading.
    OptionDescription
    Single cluster Open the ellipsis menu for the cluster and select Upgrade.
    Multiple clusters In the top-right corner, click Upgrade.
    The Upgrade wizard appears.
  7. Select when you want to run the upgrade.
    OptionDescription
    Upgrade now Initiates the upgrade right away.
    Upgrade later Provides an option to select the date and time for the upgrade.
  8. Optional: Select Rollback the failed clusters to the previous version.
    If the upgrade fails, RSC restores the Rubrik cluster to the previous Rubrik CDM version.
  9. Click Next.
    The Confirm the upgrade page appears.
  10. Review the upgrade summary and click Confirm.
  11. Click Done.

Results

RSC initiates the upgrade of the Rubrik CDM version on the Rubrik cluster. The status of the Rubrik cluster changes to Upgrade in progress.
