Events in NAS CD

You can see all the events associated with all the connected Rubrik clusters for NAS CD under the Events tab.

You can access the Event Logs and Audit Logs pages through the Events tab on the RSC Data Protection dashboard. These pages list various events in RSC.

Events Logs page

You can access the Events Logs page to see completion or failure of events such as addition of a system, deletion of a system, assignment of an SLA Domain, recovery of a snapshot, and so on.

Clicking a listed event expands it to display the associated details. Clicking View Details displays more details and provides the option to download server logs for the event.

Audit Logs page

You can see the user-initiated events in the Audit Logs page under the Events tab. You can see events and their status based on users on this page. You can download a CSV file of this page to see the details later.

Event filters

RSC tracks, monitors, and displays all the events of all the connected Rubrik clusters for NAS CD when no filter is applied. You can change the filters to change the list of Events Logs and Audit Logs that appear on the page.

The following table describes the RSC filters that you can use to filter the Event Logs view and Audit Logs view:
Filter Description
Time range

Displays events aggregated over the specified time.

  • Past 15 minutes
  • Past 2 hours
  • Past 24 hours
  • Past 7 days
  • Past 30 days
  • Custom

    With the Custom date range, you can specify a date range to include a specific number of days, including periods exceeding 30 days.
Clusters Displays events on the Rubrik cluster for NAS CD connected to RSC.
Severity

Displays the severity of the events.

  • Critical: System event that requires attention, such as failed backup, archiving, or replication.
  • Warning: System event finished with a backup, archiving, or recovery warning.
  • Informational: System event for information only.
Status

Displays events based on the event status:

  • Failed
  • In progress
  • Succeeded
  • Succeeded with warnings
  • Canceled
  • Queued
Event type

Displays events based on the specified type.

The Anomaly event type is visible and supported only when you have a valid license for Data Threat Analytics.
Object type Displays events of the specified object type related to the connected Rubrik cluster on RSC, such as NAS Cloud Direct Share, NAS Cloud Direct System, NAS Cloud Direct Bucket, and so on.
Organization Displays the events based on the specified organizations.
