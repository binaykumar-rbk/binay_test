Adding a custom intel feed

Create a custom intel feed with specific indicators of compromise to monitor for threats.

About this task

This topic describes a feature or features that Rubrik is gradually rolling out to customers.

Procedure

  1. Log in to RSC.
  2. Click and select Data Threat Analytics.
    The Data Threat Analytics dashboard appears.
  3. On the navigation bar, click Intel Feed.
    The Intel Feed page appears.
  4. Click Add Intel Source.
    The Add a Threat Intel Source wizard appears.
  5. Click Add IOC.
    The Add IOC dialog box appears.
  6. In the IOC Type dropdown list, select an option:
    • YARA rule
    • File hash
  7. Optional: In Threat Family, enter the threat family name for the IOC.
  8. In the dialog box, enter the YARA rule or comma-separated file hashes as text.
    You can add multiple IOCs.
  9. Click Next.
    The Source Details page appears.
  10. In Intel Source Name and Description, enter a name and, optionally, a description for this intel source.
  11. Click Done.

Results

Threat Monitoring adds the intel source with the IOCs to check for threats.
