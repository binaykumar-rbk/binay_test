Viewing Orchestrated Recovery event logs

The Orchestrated Recovery Event Logs page provides a chronological record of operational details during recovery events.

Procedure

  1. Log in to RSC.
  2. Click and select Orchestrated Recovery.
    The Objects page appears.
  3. On the navigation bar, hover over Events and then select Event Logs.
    The Event Logs page appears.
  4. Optional: Select filters to apply to the information on the Event Logs page.
    Orchestrated Recovery displays events based on the selected filters.
  5. Click an event.
    The event timeline and additional event details appear.

Results

Based on the specified filters, Orchestrated Recovery lists relevant events. When you select an event, Orchestrated Recovery provides historical information about that event.
