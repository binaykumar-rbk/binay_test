Performing a prioritized data recovery for SharePoint

Perform a prioritized data recovery for Microsoft SharePoint to recover only a subset of data. You can perform a prioritized data before performing a recovery of all remaining objects.

About this task

To perform prioritized data recovery, you require an Enterprise Edition license.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
  3. On the navigation bar, click Inventory.
  4. On the Inventory page, click the Microsoft 365 inventory card.
  5. From the Selected subscription dropdown list, select the Microsoft 365 subscription.
  6. Click Groups.
  7. Select one or more SharePoint groups.
  8. Click and select Recover.
    The Recover Groups wizard opens.
  9. Click Prioritized data recovery.
  10. In Recover items modified in the last x days, specify the number of days before the current date from which to recover modified items.
    For example, you can recover items from the last seven days before the current date.
  11. Optional: Select Automatically recover the remaining data after prioritized data recovery is complete to perform a recovery of the data that remains after the prioritized data recovery.
  12. Click Save, and then click Next.
  13. Select the recovery option.
    OptionDescription
    In-place recovery Recovers objects to their original location and overwirtes duplicate files.
    Recover to a new location Recovers objects to a new folder.
  14. Click Next.
  15. Select the snapshots to use to restore objects from the selected SharePoint groups.
    OptionDescription
    Use the latest snapshot Uses the latest snapshot, which is the snapshot closest in time to the time of the request.
    Use the snapshot closest to a certain time Uses the snapshot closest in time before a specified date and time.

    When you specify a Before date but do not specify the time, the snapshot used is the snapshot taken closest in time prior to 00:00:00 on the specified date.
  16. Click Recover.

Results

RSC starts a task to restore the selected data. RSC performs a full recovery if specified to do so after the prioritized data recovery.

What to do next

Recover your remaining data, if you did not specify for RSC to automatically recover the remaining data.
