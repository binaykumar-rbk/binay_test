Optional settings for adding a webhook

Information about optional settings that you can configure while adding a webhook.

Field Description
Endpoint Certificate Endpoint certificate details.

You can paste the contents of the PEM file of the public certificate belonging to your server. Additionally, you can upload a self-signed public certificate in the PEM format to establish an HTTPS connection.
Service Account Service account to filter events and audit details sent to the selected Webhook.

You can send the events and audit details to the selected Webhook, and the selected service account can view these details using RBAC.
API Key (applicable for PagerDuty and Splunk) API key for the selected web hook provider.

For Splunk, you must paste the HTTP Event Collector (HEC) token in the API Key field. The HEC token appears in HTTP Event Collector under Data Inputs in the Settings menu of your Splunk instance.
