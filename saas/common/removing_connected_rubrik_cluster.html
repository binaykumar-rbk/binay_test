Removing a connected Rubrik cluster

Remove a connected Rubrik cluster from RSC if it is no longer required.

Before you begin

  • Make sure all prerequisite tasks are completed successfully. Information about prerequisites is included in Remove a Rubrik cluster.

  • If you've enabled quorum authorization, you must create a quorum authorization policy before you can initiate the removal of a Rubrik cluster. After you initiate the removal, a quorum authorization request is automatically submitted. The cluster removal proceeds only after the required approvals are granted. Quorum authorization provides more information about quorum authorization.

Important:
The Rubrik cluster removal process is irreversible. Once initiated, all data related to the Rubrik cluster is permanently deleted.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
  3. On the navigation bar, click Infrastructure > Clusters.
  4. Click a Rubrik cluster.
    The Cluster page appears.
  5. Click and then select Remove.
    Alternatively, on the line for the cluster on the Clusters page you are removing, click next to the cluster you want to remove and select Remove.
    The Remove Cluster wizard appears and displays the Cluster Details page.
  6. Review the details of the Rubrik cluster.
    The status of these tasks is shown under Review Prerequisites for Removal. If prerequisite tasks exist, a message appears identifying the tasks you must complete before proceeding. If no prerequisite tasks exist, a message appears confirming that the cluster has no active dependencies and can be removed.
  7. Click Next.
  8. Enter CLUSTER to confirm the action to remove the cluster.
  9. Click Remove.

Results

The process to remove the Rubrik cluster starts. This process may take between 30 minutes to 24 hours to complete.

While this task is in progress, the status of the Rubrik cluster is shown as Disconnecting. Once complete, all data in RSC related to this cluster is deleted, and the cluster is no longer shown on the Clusters page.

The cluster removal event is displayed on the Events page. The Audit Log displays the corresponding message.
