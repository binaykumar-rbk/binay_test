Webhook payload

Information about the common event and audit payload formats that RSC sends to a webhook endpoint.

The following table describes the common fields in an event or audit payload.
Field Description Type Example
Summary A high-level text summary message of the event or audit. String Successfully completed preparation of t_6984 taken at 2022-07-20 21:41 (UTC). Visit the Download Center to obtain the link to download the report.
Source Specific human-readable unique identifier.

This field consists of the Rubrik cluster ID or the RSC cluster ID and the protected object ID that generated the event.

 String 1f513cf1-32cf-4906-67ac-de4f96e49c93 | ManagedVolume:::<UUID>

The UUID is 32 hexadecimal characters and four hyphens. An RSC cluster ID starts with a zero.
Severity The severity of the impact to the affected system. Enum {Info, Warning, Error, Critical} Info, Warning, Error, Critical
Timestamp The time when RSC detected or created the event or audit.

This is useful for determining if batches or hold events occurred before sending them to the server or application.

 Timestamp May 26 at 12:55 PM
Class Class or type of the event or audit. String Diagnostic, Backup, Index, Configuration
Custom Custom fields for the event or audit. Not applicable Not applicable
The following table describes the custom fields in an event or audit payload in RSC.
Field Description Type Example
SeriesID The top-level ID of an operation.

All event IDs belong to a unique series ID. For example, if a backup operation is initiated, the associated series ID may consist of multiple events.

 String 2955cde9-87g3-7j63-ddas-43jd55834ndf
ID Unique ID of the audit entry that RSC generated for the event. String 2986cde9-57g3-4g63-djks-43jd58934ggs
Type Type of the payload. String Event, Audit
ObjectID Unique ID of the object associated with the event. String auth0|7c00d5cec4caef20cf97854d5
ObjectName Name of the object. String john.doe@rubrik.com, <protectable object name>
ObjectType Type of the object. String User, ManagedVolume, SQL server DB
Status Status of the event. String Success, Failure
ClusterID ID of the cluster on which the event occurred. String <UUID>

The UUID is 32 hexadecimal characters and four hyphens. RSC clusters have a UUID that is all zeros.
ClusterName Name of the cluster on which the event occurred. String BA-Edge1
EventName A short name for the event. String Snapshot.BackupSucceeded
ErrorID A unique ID of the error. String Vmware.VsphereSnapshotAlreadyExists
ErrorCode Error code for the failed operation. String RBK20100027
ErrorReason Information about the reason of the error. String Cannot take vSphere snapshot as there are too many snapshots. Taking an additional snapshot will cause nested snapshots, which affects vSphere performance.
ErrorRemedy Information about the resolution of the error. String Using the vCenter snapshot manager, remove or consolidate any existing snapshots before performing another snapshot operation.
Location Location of the object. String iad-prd-support-vc001.iad.support.rubrik.com
URL Custom URL of the event. This field is optional. String URL to download a file from an event in RSC.
CustomerID ID of the RSC customer account. String rubrik-tme
auditUserName Name of the user who initiated the operation associated with the audit. This field is applicable only for the audit payload. String john.doe@rubrik.com
auditUserID ID of the user who initiated the operation associated with the audit. This field is applicable only for the audit payload. String auth0|7c00d5cec4caef20cf97854d5
logicalSizeInBytes The total logical size of the data transferred, such as the logical size of the snapshot.

This field is only applicable for data transfer-related operations. Otherwise, it is null.

 String 300
dataTransferredInBytes The amount of data transferred in bytes if the associated workflow involves data transfer. If the workflow does not involve data transfer, this field is null. String 100
effectiveThroughput The effective throughput of data transfer associated with the event till now. The unit is bytes per second. String 200
