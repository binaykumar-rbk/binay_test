Viewing audit log information

RSC displays the Audit log page and lists user-initiated events by user names.

Procedure

  1. Log in to RSC.
  2. On the navigation bar, click Events.
    The Events page appears.
  3. Click Audit log.
    The Audit Log Details page appears.
  4. Optional: Select filters to apply to the Audit log page.
    RSC populates the Audit Log Details page with the aggregate values based on the selected filter.

Results

RSC displays the Audit log page.
