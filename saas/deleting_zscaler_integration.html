Deleting a Zscaler integration

Delete the Zscaler integration from RSC if it is no longer required.

About this task

This topic describes a feature or features that Rubrik is gradually rolling out to customers.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Integrations.
    The Integrations page appears.
  4. Click the Zscaler integration card.
    The Zscaler page appears.
  5. Open the ellipsis menu for a Zscaler integration and select Delete.
    A confirmation dialog box appears.
  6. Click Delete.

Results

RSC deletes your Zscaler integration.
