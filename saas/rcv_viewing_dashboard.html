Viewing the RCV dashboard

View the Rubrik Cloud Vault dashboard for backup and archive summary information related to all the Rubrik Cloud Vault locations connected to RSC.

About this task

Viewing important information about all the Rubrik Cloud Vault (RCV) locations on one dashboard helps you to quickly monitor the progress and analyze the problems related to backup and archive tasks at different locations.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Infrastructure > Rubrik Cloud Vault.
    The Rubrik Cloud Vault dashboard appears.

Results

RSC displays a comprehensive overview of the RCV locations.
