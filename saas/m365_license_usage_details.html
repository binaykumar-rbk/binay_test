Microsoft 365 license and usage details

The Microsoft 365 Licenses and Usage page provides detailed information about the purchased licenses and consumption details across various Microsoft 365 services.

The following table describes the charts available on the Microsoft 365 Licenses and Usage page.
Chart/Table Description
Users Protected/Purchased Displays the number of user licenses purchased in the last six months along with the number of utilized licenses. The White line indicates the number of user licenses purchased. The Blue line indicates the number of protected users.
Capacity Used/Purchased

Displays the data capacity purchased in the last six months along with the utilized capacity. The White line indicates the purchased capacity. The Blue line indicates consumed capacity.

RSC does not display this chart for only user-based licenses.
Usage Distribution Displays the breakdown of capacity usage.
Licenses Displays a list of all the RSC licenses purchased for the workload. Based on the expiration date, RSC uses the following indicators to help track licenses:

  • A warning sign in the Term column indicates that the license expires in the next seven days.

  • A caution sign in the Term column indicates that the license expires in the next three months.

  • An information sign in the Term column indicates that the license expires in the next six months.

  • A Scheduled label next to a license name indicates that the license is purchased for future use.

The table also displays the licenses that expired in the last six months.
