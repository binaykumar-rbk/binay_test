Failover cluster management actions

RSC provides the ability to edit and delete failover clusters.

RSC provides an ellipsis menu for each failover cluster and a top-level ellipsis menu for all the management actions related to the clusters.

Management action Description
Ellipsis menu > Edit Failover Cluster Allows edits to the failover cluster name and hosts that comprise the cluster.

The Rubrik cluster associated with the failover cluster cannot be edited.
Ellipsis menu > Delete Failover Cluster Deletes the failover cluster from the Rubrik cluster and removes protection for all the services associated with the failover cluster. All existing snapshots are managed as relics.
Top-level ellipsis menu > Delete Failover Clusters Deletes multiple failover clusters from the Rubrik cluster and removes protection for all the services associated with the failover clusters. All existing snapshots are managed as relics.
