Data Threat Analytics pane

The Data Threat Analytics pane of the Global Dashboard provides several interactive elements.

Information type Description
Pipeline health The overall health of the Data Threat Analytics pipeline. Pipeline health represents the overall success of backup, indexing, and analysis jobs in the last 24 hours.
Recently scanned object total Displays the total number of objects Data Threat Analytics has scanned in the last 24 hours.
Recent anomalies Displays the number of anomalies detected in the last 24 hours.

Click the investigate button to navigate to the Events page, with the page view filtered to show only anomalies.
