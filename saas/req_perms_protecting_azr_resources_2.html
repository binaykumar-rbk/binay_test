          You can set up SLA Domains to apply data protection policies, ensuring automatic and hierarchical protection of GitHub organizations and repositories, with granular control overriding higher-level assignments.

          After completing the protection setup for GitHub organizations and repositories, RSC takes automated snapshots of those repositories based on the configured policies in SLA Domains. You can also take on-demand snapshots and refresh GitHub organizations for better resource management.
