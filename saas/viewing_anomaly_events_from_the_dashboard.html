Viewing anomaly events from the Dashboard

RSC provides detailed information about detected anomaly events in the Data Threat Analytics dashboard.

Procedure

  1. Log in to RSC.
  2. Open the app tray and select Data Threat Analytics.
    The Data Threat Analytics dashboard appears.
  3. Click the Anomalies widget.
    The Anomaly Detection page appears.

    VM Encryption Detection anomalies are indicated as Ransomware - Hypervisor.

  4. Click an object name to view anomaly details.

Results

The Anomaly event details page appears.
