Editing SLA Domains for NAS CD

Edit an existing NAS CD SLA Domain to update its details.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click SLA Domains.
    The SLA Domains page appears.
  4. Click an SLA Domain to modify.
    The SLA Domain details page appears.
  5. Click Edit.
    The Edit SLA Domain wizard appears with the object type selected.
  6. Click Next.
  7. Optional: Modify the Retention Lock settings and select a mode, if applicable.
    ModeDescription
    Governance mode Requires quorum authorization for reducing the SLA Domain retention period.
    Compliance mode Does not support assignment or reduction of the retention of this SLA Domain. Snapshots do not expire before the scheduled expiration date.

    With this mode, you need to select the mandatory acknowledgment for proceeding to the next step.
    Turn on retention lock if you want to ensure security and integrity of your data for a specified duration by preventing the premature deletion of the snapshot.
  8. Click Next.
  9. Edit the snapshot frequency, corresponding retention period, and then click Next.
    The snapshot frequency categories are optional, but at least one category must have a value.
    The Archival and Replication page appears.
  10. Click Next.
    As NAS CD does not support Archival and Replication, these features are not enabled in RSC.
    The Object-Specific Settings page appears.
  11. Click Edit NAS CD Site Selection and edit the Rubrik cluster selection.
    You can select multiple clusters.
  12. Click Save.
  13. Select a target for each of the Rubrik cluster you selected.
    If you cannot find the target location you want, create the target location from the Archival and Storage page in RSC. The Creating targets topic provides more details about the creating NAS CD targets.
  14. Click Next.
    The Confirmation page appears.
  15. In SLA Domain Name, edit the name of the SLA Domain.
  16. Optional: In Description, edit the description for the SLA Domain.
  17. Review the changes in the configuration details and click Save.

Results

You have successfully edited the details for the SLA Domain.
