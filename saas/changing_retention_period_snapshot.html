Changing the retention period of a snapshot

Change the retention period of a snapshot of data center or cloud native workload objects.

About this task

You can change the retention period of on-demand snapshots with an SLA Domain or a Forever retention policy assigned.

You cannot change the retention period of policy-based snapshots with SLA Domain, without an SLA Domain, or with Forever retention policy assigned.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Snapshots and select Retention.
    The Snapshot Retention page appears.
  4. Select one of the following to display the objects associated with a specific Rubrik cluster or a cloud native workload:
    OptionDescription
    Data Center Filter the objects by Rubrik cluster. You can sort the objects based on the following filters:
    • Rubrik Cluster
    • Object Type
    • Object Status
    • SLA Domain
    Cloud Filter the objects managed by a specific cloud native workload. You can sort the objects based on the following filters:
    • Managed by
    • Cloud Account
    • Region
    • Object Type
    • Object Status
    • SLA Domain
  5. In the line corresponding to the object name, click the number listed in the Snapshots column.
    The Snapshots page displays the list of the snapshots for the selected object.
  6. Select a snapshot and click Change Retention.
    The Change Retention wizard appears.
  7. Select a retention policy.
    Retention policyDescription
    SLA Domain The snapshots are retained on all locations for the maximum retention period specified by the chosen SLA Domain.
    Forever The snapshots are retained until they are manually deleted.
  8. Click Next.
    A summary information describing the impact of the changes appears.
  9. Optional: In Add a note, enter text to attach to the object type for this task.
    You can enter notes up to 1024 characters.
  10. Review the summary information and click Assign.

Results

RSC updates the retention period of the selected snapshots.
