            A data source is designated as a relic if it is managed by, but is no longer accessible to the Rubrik cluster.

          • Intelligent Data Lock

          Legal hold ensures that snapshots are retained indefinitely until the hold is removed, overriding the scheduled expiry of an SLA Domain policy. This ensures that snapshots are available as long as necessary to meet legal obligations.

Searching for an object on the Snapshot Retention page

Search for a specific object by object name on the Snapshot Retention page.

Procedure

  1. Log in to RSC.
  2. Click and select Settings.
    The Settings menu appears.
  3. Click Snapshots and select Retention.
    The Snapshot Retention page appears.
  4. In the search box, enter the name of the object.
    The specified object appears.

Results

RSC searches for the specified object and displays it on the Snapshot Retention page.

What to do next

You can select the object and choose to change the retention period or delete the associated snapshot.
