Anomaly Detection details

The Anomaly Detection details page provides detailed information about the anomaly and the object it was indicated on, files changed within the snapshot, and the event timeline.

The Anomaly Detection details page provides information about the protected workload in the Summary tab and the Browse tab. You can download the file with these details by clicking the Download CSV file button.

Tab Description
Summary The Summary tab shows anomalous activity over the past seven days and the total number of suspicious, deleted, added, and modified files for the protected object. The following widgets are available in the Summary tab:
  • Anomaly Details: Provides information on the Detection Time, Encryption, and Severity.
  • Data Sensitivity: Provides a high-level summary of sensitive data residing on the affected object. This information is retrieved from Data Security Posture and is categorized by Hits by Sensitivity and Hits by Data Category.
  • Activity: Shows anomalous activity over the past seven days.
  • Object Details: Provides information about the protected object.
  • Event Timeline: Provides messages for each event in the anomaly event series.
  • Suspicious Files: Provides the path to the suspicious file, the suspicious activity on the file, the date it was last modified, and an option to download files for forensic analysis.

    You can use the Download for Forensics option to download files from each suspicious activity for further analysis. Suspicious activities are categorized into two types: Ransomware note and Ransomware encryption. Ransomware note refers to files that match the characteristics associated with ransom notes left by attackers. Ransomware encryption refers to files with metadata that suggest possible ransomware encryption.

    Note:
    Ransomware encryption suspicious activity refers to the files that ransomware modifies, not the malware itself.
Browse The Browse tab displays information about the changes in the file system and provides a comparison between the selected snapshot and the previous snapshot. The following information is available in the Browse tab:
  • Changes in the number of files that are added, modified, or deleted
  • Changes in size of the protected resources, the total size of the protected resource
  • Total size of the protected resource
  • Time stamp when the protected resources were last modified

You can complete the following actions from the Browse tab:

  • Viewing anomaly details
  • Sorting data by the Suspicious or File Changes filters
  • Searching for objects using the search bar
  • Recovering a protected object
  • Viewing the object or files in Data Security Posture. This is only available if Data Security Posture has been enabled for the deployment and the object.

You can perform the following actions from the Summary and the Browse tabs:

  • Viewing older anomalies of the object
  • Downloading a CSV file of the anomalies in that object. When you click Download CSV, RSC prepares a secure link for the file and displays a message for download. You can access the downloaded file from the File Preparation Center.
  • Reporting a false positive anomaly
  • Resolving an anomaly
  • Recovering a full snapshot of the object
