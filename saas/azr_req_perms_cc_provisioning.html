      • Rubrik Cloud Cluster ES for Azure

        You can use Rubrik Cloud Cluster Elastic Storage for Azure to protect applications and data within a Microsoft Azure virtual network.

        • Prerequisites for Rubrik Cloud Cluster ES on Azure

          Information about the prerequisites for provisioning Rubrik Cloud Cluster Elastic Storage.

        • Node requirements for Rubrik Cloud Cluster on Azure

          RSC uses a cloud account that requires certain permissions or actions to provision Rubrik Cloud Cluster Elastic Storage for Azure.

        • Transitioning to using Azure managed identities

          For enhanced security, transition your cluster from using connection strings to using managed identities as the authentication type to your storage container.

        • Workflow for Rubrik Cloud Cluster ES for Azure

      • Rubrik Cloud Cluster ES for AWS

        A Rubrik Cloud Cluster Elastic Storage protects applications and data by running Rubrik CDM software on Amazon EC2 instances within a virtual private cloud.

      • Rubrik Cloud Cluster recoverability

        To restore your data, you can recover a Rubrik Cloud Cluster Elastic Storage that has immutability enabled. You can do this only when the cluster nodes cannot be recovered.

      • Adding a node to Cloud Cluster ES

        Add a node to a Cloud Cluster ES to increase capacity and improve performance.

      • Adding a pre-deployed node to Rubrik Cloud Cluster ES

        Add a node that you have manually deployed in AWS or Azure to the Rubrik Cloud Cluster ES in RSC. Additional nodes increase capacity and improve performance.

      • Node removal in RSC

        If you no longer require a node in a Rubrik cluster, a supported third-party hardware cluster, or a Rubrik Cloud Cluster, you can remove the node by using RSC. To remove a node, you must meet the requirements for node removal.

      • Extra dense node migration for CCES clusters

        Extra dense nodes offer greater storage and performance, without a loss of storage capacity. Using extra dense nodes, organizations can reduce node counts by half during migration. Customization options are available through the GraphQL API.

Required permissions for Azure Cloud Cluster provisioning

RSC uses a cloud account that requires certain permissions or actions to provision Rubrik Cloud Cluster Elastic Storage for Azure.

The following table provides information about the actions.
Action Description
Microsoft.Compute/disks/read Retrieves the properties of a disk.
Microsoft.Compute/disks/write Retrieves virtual machines.
Microsoft.Compute/disks/delete Reverts a virtual machine export task.

Microsoft.Compute/disks/beginGetAccess/action

Microsoft.Compute/disks/endGetAccess/action

 Starts or stops the cross-region export of a virtual machine.
Microsoft.Compute/locations/vmSizes/read Lists the available virtual machine sizes in a particular region for export.
Microsoft.Compute/skus/read Lists the available virtual machine sizes in a particular region for export.
Microsoft.Compute/snapshots/read Lists the properties of a snapshot.
Microsoft.Compute/snapshots/write Takes snapshots.
Microsoft.Compute/snapshots/delete Retries or reverts a virtual machine or managed disk snapshot task.

Microsoft.Compute/snapshots/beginGetAccess/action

Microsoft.Compute/snapshots/endGetAccess/action

 Performs the cross-region export of a snapshot.
Microsoft.Compute/virtualMachines/delete Exports a virtual machine.
Microsoft.Compute/virtualMachines/instanceView/read Retrieves the detailed runtime status of the virtual machine and its resources.

Microsoft.Compute/virtualMachines/powerOff/action

Microsoft.Compute/virtualMachines/restart/action

Microsoft.Compute/virtualMachines/start/action

 Restores a virtual machine.
Microsoft.Compute/virtualMachines/read Retrieves the properties of a virtual machine.
Microsoft.Compute/virtualMachines/write Exports a virtual machine.

Microsoft.Network/applicationSecurityGroups/joinIpConfiguration/action

Microsoft.Network/applicationSecurityGroups/joinNetworkSecurityRule/action

Microsoft.Network/applicationSecurityGroups/listIpConfigurations/action

Microsoft.Network/applicationSecurityGroups/read

 Associates Application Security Groups in a virtual machine export task.
Microsoft.Network/networkInterfaces/delete Exports a virtual machine.

RSC creates a network interface and deletes it when reverting the export task.
Microsoft.Network/networkInterfaces/ipconfigurations/read Retrieves the subnet details of a virtual machine during the refresh task.
Microsoft.Network/networkInterfaces/join/action Attaches a network interface to a virtual machine during the Export virtual machine task.
Microsoft.Network/networkInterfaces/read Retrieves a network interface definition when performing write operations.
Microsoft.Network/networkInterfaces/write Creates a network interface during the Export virtual machine task.
Microsoft.Network/networkSecurityGroups/join/action Associates a Network Security Group (NSG) with a network interface during the Export virtual machine task.
Microsoft.Network/networkSecurityGroups/read Retrieves the NSGs to populate the dropdown list in RSC during the Export virtual machine task.
Microsoft.Network/virtualNetworks/read Retrieves the Azure Virtual Network (VNet) details during the Export virtual machine task.
Microsoft.Network/virtualNetworks/subnets/read Retrieves the details of virtual network subnets during the Export virtual machine task.
Microsoft.Network/virtualNetworks/subnets/join/action Creates a virtual machine in a subnet during the Export virtual machine task.
Microsoft.Resources/checkResourceName/action Checks the resource name for validity when creating resources.
Microsoft.Resources/subscriptions/locations/read Lists only those regions for which the subscription is configured.
Microsoft.Resources/subscriptions/resourceGroups/read Refreshes tasks.

Microsoft.Resources/subscriptions/resourceGroups/resources/read

Microsoft.Resources/subscriptions/resources/read

 Optimizes refresh tasks.
Microsoft.Resources/subscriptions/resourceGroups/write Creates a separate resource group to store snapshots taken by RSC.
Microsoft.KeyVault/vaults/deploy/action Exports Azure Disk Encryption (ADE) enabled virtual machines and disks.
Microsoft.Compute/virtualMachines/extensions/write Enables encryption on an ADE-enabled virtual machine.
Microsoft.Authorization/locks/* Locks resources.
Microsoft.Network/networkSecurityGroups/write Creates an NSG when creating a powered-off virtual machine during the virtual machine export task.
Microsoft.Network/networkSecurityGroups/delete Deletes an RSC-owned NSG when creating a powered-off virtual machine during the virtual machine export task.
Microsoft.Storage/storageAccounts/write Creates a storage account in a region during the cross-region replication.
Microsoft.Storage/storageAccounts/listServiceSas/action Generates Shared Access Signatures (SAS) Uniform Resource Identifier for reading, writing, creating, and deleting page blob and its snapshots during the cross-region replication.
Microsoft.Storage/storageAccounts/blobServices/containers/{read, write, delete} Creates, writes, and deletes page blob containers during the cross-region replication.
Microsoft.Compute/availabilitySets/read Lists availability sets to which a virtual machine can be exported.
Microsoft.Storage/storageAccounts/read Lists storage accounts for file-level recovery.
Microsoft.Compute/diskEncryptionSets/read Exports virtual machines across regions, using a different SSE-CMK key for exported virtual machines.
Microsoft.Compute/galleries/images/versions/read Restores and exports virtual machines with linked gallery images.
Microsoft.Storage/storageAccounts/listkeys/action Accesses the storage account for Rubrik Cloud Cluster Elastic Storage (ES) for Azure. Rubrik Cloud Cluster ES for Azure needs a storage account key during bootstrap.
Microsoft.ManagedIdentity/userAssignedIdentities/read Lists the user-assigned managed identities to associate with Azure virtual machines for Rubrik Cloud Cluster ES for accessing Azure storage.
Microsoft.ManagedIdentity/userAssignedIdentities/assign/action Attaches the user-assigned managed identities to virtual machines for Rubrik Cloud Cluster ES.
Microsoft.Compute/images/read Retrieves image names when deploying Azure Cloud Cluster virtual machines.
Microsoft.Insights/metrics/read Extracts metrics from Azure virtual machines.
Microsoft.MarketplaceOrdering/offerTypes/publishers/offers/plans/agreements/read Retrieves agreements for subscribing to Azure marketplace offerings.

Permission Description
