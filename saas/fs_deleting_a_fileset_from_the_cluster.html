          • Fileset fields, rules, and value types

            The Rubrik cluster interprets a fileset based on the values provided in the Include, Exclude, and Do Not Exclude fields. The Rubrik cluster applies a set of rules to the values provided in these fields and permits several types of values to be added to the fields.

          • Host-specific fileset description rules

            Rules for fileset descriptions differ for host types, ensuring accurate file path configurations, preventing backup errors, and optimizing backup processes for different systems.

          • Accepted values for fileset descriptions

            Rubrik clusters accept a specific set of values in fileset descriptions.

          • Fileset description rules

            Fileset descriptions must meet specific requirements.

          • Regular expression handling

            Fileset rules convert to regular expressions that follow the Boost regular expression syntax guidelines.

          • Fileset regular expression conversions

            The Rubrik cluster converts regular expressions used during the specification of a fileset rule.

          • Fileset details

            For every fileset, RSC displays details including the fileset rules and the number of hosts being protected through the fileset.

Deleting a fileset from the Rubrik cluster

Delete a fileset from the Rubrik cluster to remove all existing associations of that fileset with the hosts that use the fileset to protect files and folders on the hosts.

Before you begin

Before deleting a fileset, make sure that the associated Rubrik cluster is in a connected state.

About this task

RSC provides the option to move the host fileset and all associated backups to the Snapshots tab of the host or to permanently delete the fileset and all associated backups.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click a fileset card based on the OS type of the host.
    • Windows filesets
    • Linux & Unix filesets
    The corresponding filesets page appears.
  5. Open the Filesets tab.
  6. Select a fileset.
    Multiple filesets associated with the same Rubrik cluster may be selected.

    Filter the filesets by the Rubrik cluster where they were created or search for a fileset by name.

  7. Open the top-level ellipsis menu and select Delete filesets.
    The Delete filesets dialog box appears.
  8. (For assigned filesets) Choose how to handle the existing backups of all associated host filesets.
    OptionDescription
    Transfer snapshots to relic Moves the fileset and associated backups to the Snapshots tab.
    Expire snapshots immediately Permanently deletes the fileset and all associated backups.
  9. Click Delete.

Results

RSC deletes the fileset from the Rubrik cluster and handles the backups as specified.
