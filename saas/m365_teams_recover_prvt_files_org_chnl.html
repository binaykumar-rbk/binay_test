Recovering Teams private channel files to the original channel

Use indexed Microsoft 365 snapshots to recover files uploaded to a Microsoft Teams private channel to the original channel.

Before you begin

For private channel file recovery, confirm that the Teams app is authenticated. If the Teams app is not authenticated, re-authenticate the Teams app for the tenant.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
  3. On the navigation bar, click Inventory.
  4. Click the Microsoft 365 inventory card.
  5. From the Selected Subscription dropdown menu, select the subscription.
  6. Click Enter Subscription.
  7. Click Teams and select a team.
  8. On the details page for the team, select a snapshot from the calendar view and click Recover.
    Alternatively, you can search for an object to recover.
  9. Click Recover selected content for one channel type and then click Next.
  10. Click Files > Private Channels and then click Next.
  11. Select a private channel and then click Next.
  12. Select the files and folders to recover and then click Next.
  13. Select Recover to original channel and then click Next.
  14. Select the recovery option.
    OptionDescription
    In-place recovery Recovers files to their original location in the selected directory. In-place recovery recovers only objects in folders/files/attachments.

    In-place recovery of conversations is not supported.
    Recover to a new folder Recovers files to a new folder in the target location.
  15. For in-place recovery, select a recovery option.
    OptionDescription
    Append duplicate objects Creates duplicates of the objects and retains the original objects. RSCcreates the duplicates at the file level, not at the folder level.
    Overwrite duplicate objects With duplicate objects, RSC overwrites the old objects with the objects to be recovered. RSC overwrites files at the object level, not at the folder level.
  16. Click Next.
  17. Log in to the Microsoft 365 account to provide RSC with the permissions required for recovering objects to the selected channel and then click Next.
  18. Review the information and then click Recover.

Results

RSC schedules a task to complete the requested recovery. After recovering to a new folder, the restored files appear in the Files tab of the original channel with the filename RBK_YYYY_MM_DD.

For in-place recovery, files are recovered to their original location.

When recovered to ZIP format, the file download link appears in the Events timeline.
