Downloading the upgrade package from the Support Portal

Use the Rubrik CLI to download the Rubrik CDM software upgrade package and the associated signature file from the Rubrik Support Portal to the same location on a node of the Rubrik cluster.

Procedure

  1. Log in to the Rubrik CLI using the administrator account.
  2. At the prompt, enter cluster upgrade download_and_precheck from_rubrik_release_repository.
  3. Enter a username and password for the Support Portal, if prompted.
    Provide the username and password of an account that has permission to view and download the upgrade package.
    A list of compatible Rubrik software versions appears.
  4. Select a software version.
    A progress bar shows the progress of the job running in the background.
    The compressed files are automatically extracted during this step.
  5. Enter cluster upgrade status watch to monitor the status of the download and upgrade prechecks operations.
  6. Enter cluster upgrade status to view the status of the download and upgrade prechecks operations once these operations are completed.
    The upgrade state must display as STAGED.

    The upgrade prechecks status can be Passed or Failed. If the status is Failed, you must resolve the issues and rerun the prechecks using the cluster upgrade start --mode=prechecks_only command.

Results

The Rubrik cluster downloads the upgrade package, extracts the files from the package, and pre-stages the files for upgrade.
