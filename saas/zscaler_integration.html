Zscaler integration for data loss prevention

You can protect your sensitive data by integrating Zscaler with RSC. You can identify sensitive data objects and prevent any unauthorized transfer of your most critical data by leveraging the Rubrik Data Security Posture and Zscaler Data Loss Prevention services.

This topic describes a feature or features that Rubrik is gradually rolling out to customers.

Rubrik Data Security Posture service integration with Zscaler Data Loss Prevention (DLP) proactively identifies sensitive business data across enterprise, cloud, and SaaS environments so that you can implement specific data protections to prevent data loss.

The Zscaler DLP feature uses existing functionality in RSC and Zscaler software to ensure the security of sensitive data in your production environment. This feature prevents unauthorized data transfers, known as exfiltration, by identifying files with sensitive data through Rubrik Data Security Posture service and exporting them to a drop target machine scanned by Zscaler indexer software.

From the drop target machine, the indexer software fingerprints the files, generates a unique hash value for each file, and uploads this hash value to the Zscaler cloud. Zscaler cloud blocks any attempt to copy the files to a location outside your production environment, such as Dropbox or a USB, thus keeping your files secure and protected.

The following diagram describes an overview of the Zscaler integration.

  1. Rubrik CDM creates snapshots.
  2. Rubrik Data Security Posture (DSP) scans for sensitive data according to assigned policies.
  3. Zscaler DLP lists files with sensitive data according to the selected policies.
  4. Zscaler DLP triggers the export of files with sensitive data.
  5. Rubrik CDM exports files with sensitive data.
  6. Zscaler Indexer fingerprints files.
  7. Zscaler Indexer uploads fingerprints to Zscaler cloud.
