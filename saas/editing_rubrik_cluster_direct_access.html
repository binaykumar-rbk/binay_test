Editing a Rubrik cluster for direct access

Edit a Rubrik cluster to specify an IP address or a fully qualified domain name to be used for direct cluster access.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
  3. On the navigation bar, click Infrastructure > Clusters.
  4. Click the name of the Rubrik cluster to edit.
    The cluster details page appears.
  5. Open the menu and select Edit.
    The Edit Cluster dialog box appears.
  6. In IP or FQDN, enter the IP address or fully qualified domain name (FQDN) of the cluster.
    If left unspecified, RSC automatically selects to use the management IP address of one of the nodes in the cluster for direct cluster access.
  7. Click Save.

Results

You have now configured the Rubrik cluster for direct access using the specified IP address or FQDN.
