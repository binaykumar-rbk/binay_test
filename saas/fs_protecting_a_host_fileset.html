Protecting a host fileset

Create a host fileset by pairing a fileset with a host. Assign an SLA Domain to the host fileset to protect the data in the host fileset.

Before you begin

Add the Linux, Unix, or Windows host to the Rubrik cluster, as described in Adding a fileset host.

About this task

To manage protection of a host on a Rubrik cluster, the Rubrik cluster must be in a connected state.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Inventory.
    The Inventory page appears.
  4. Click a fileset card based on the OS type of the host.
    • Windows filesets
    • Linux & Unix filesets
    The corresponding filesets page appears.
  5. Select a host.

    You can select multiple hosts associated with the same Rubrik cluster to apply a fileset and protect all the host filesets with the same SLA Domain.

  6. Click Manage Protection.
    The Manage Protection wizard opens the Select fileset page.
  7. Optional: Click the + icon to create a fileset.
    After creating a fileset, the Select fileset page displays the new fileset.
  8. Select a fileset.
  9. Click Next.
    The Manage Protection wizard progresses to the Assign SLA Domain page.
  10. Optional: Click the + icon to create an SLA Domain.
    The Assign SLA Domain page displays the new SLA Domain.
  11. Select SLA Domain.
  12. Optional: Select Direct Archive to enable Direct Archive for the fileset.

    Direct Archive is only available when the fileset is assigned to an SLA Domain that specifies an archival location.

  13. Optional: (Linux and Unix) Select Symlink to resolve symbolic links for Linux and Unix filesets.
    Enabling this option could increase the backup time for Linux and Unix filesets.

    If a symbolic link is not resolved, it appears as a file, rather than as a folder, in the UI. Symbolic links in Linux and Unix filesets that point to a directory in a different fileset are not resolved.

  14. Click Next.
    The Manage Protection wizard progresses to the Confirmation page.
  15. Optional: In Add a note, enter text to attach to the object type for this task.
    You can enter notes up to 1024 characters.
  16. Confirm that the Frequency and Retention settings are correct and click Assign.

Results

RSC assigns the selected SLA Domain to the host filesets and manages the protection on the Rubrik cluster.
