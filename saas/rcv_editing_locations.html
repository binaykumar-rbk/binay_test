Editing RCV locations

Edit a Rubrik Cloud Vault location to modify the location name, bypass proxy setting, or IP address allowlist.

Procedure

  1. Log in to RSC.
  2. Click and select Data Protection.
    The Data Protection dashboard appears.
  3. On the navigation bar, click Infrastructure > Rubrik Cloud Vault.
    The Rubrik Cloud Vault dashboard appears.
  4. Click RCV Locations.
    A list of RCV locations appears.
  5. Click and select Edit Rubrik Cloud Vault.
    The Edit Rubrik Cloud Vault page appears.
  6. Make the required changes and click Next.
    Based on the RCV location, you can edit the following fields:
    • For RCV locations hosted on Azure, you can edit the location name, bypass proxy setting, redundancy type, and IP address allowlist for data center archival locations. For cloud-native and NAS CD archival locations, you can edit the location name and redundancy type.
    • For RCV locations hosted on AWS, you can edit the location name and bypass proxy setting for data center archival locations. For cloud-native and NAS CD archival locations, you can only edit the location name.
    • For RCV locations hosted on GCP, you can only edit the location name.
    The Confirmation page appears.
  7. Review the changes and click Update.

Results

You have successfully updated the RCV location.
