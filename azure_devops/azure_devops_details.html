Azure DevOps details

RSC monitors, analyses, and restores Azure DevOps repositories associated with the Azure DevOps projects and organizations. RSC also shows the important details such as organization status, protected projects, and protected repos, and summary of backup jobs.

Azure DevOps Inventory tab details

The following table provides information about tabs available on the Azure DevOps inventory page.
Tab Description
Summary Provides a quick overview of your protected Azure DevOps environment. This tab is the main dashboard for assessing the health and connection status of your organizations and the performance of your backup operations. You can view the organization status, the total number of connected projects and repos, and the percentage of successful and failed backup jobs on this tab.

You can add new organizations and view system events.
Projects Lists all projects across your connected Azure DevOps organizations. This enables you to manage protection at the project level, which is a container for repositories within an organization.
Repos Lists all repos across your projects along with details such as the assigned SLA Domain, organizations, and projects. You can also view snapshots associated with each repo. You perform the data protection and recovery operations on this tab.
Organizations Lists all added organizations and their assigned SLA Domain. You can refresh organizations to ensure data synchronization, manage protection at the organization level, search for an organization, and sort the list by the assigned SLA Domain. The Organizations tab is the central point for managing the high-level connection and protection status of your Azure DevOps organizations within RSC.

Summary tab details

The following table provides information about panes available on Summary tab of the Azure DevOps inventory page.
Pane Description
Organization Status Displays the details associated with connection and permission health of your Azure DevOps organizations. You can view the Organization Status pane to quickly identify the number of organizations that are connected successfully and the ones that are missing necessary permissions for RSC to perform backup operations.
Projects Displays the number of projects that RSC protects.
Repos Displays the number of repos that RSC protects. The Repos pane also shows the total backup data and the date and time of the last backup.
Backup Jobs Displays the processing status of the automated data protection policies configured for your repos through the SLA Domains. The Backup Jobs pane displays a percentage of successful and failed backup jobs, providing an immediate performance indicator of your protection setup.
