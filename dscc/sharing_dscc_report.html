Sharing a Data Security Command Center report

A Data Security Command Center report lets you check how ready your organization is for data security. You can also see suggestions to improve low data security scores, helping you to make the right decisions to boost data security.

Procedure

  1. Log in to RSC.
  2. Open the app tray and select Data Security Command Center.
    The Data Security Command Center dashboard appears.
  3. Click Share Report.
    The Share Report window appears.
  4. To identify each intended recipient of the report, in Recipients, click + and enter their email address.
    RSC adds the email address of the sender by default.
  5. In Set a Password, enter a password.

    Ensure that you adhere to the password requirement standards. For example, the password must have a minimum of 10 characters and must include at least one capital letter and one special character. To set a strong password, use uncommon phrases.

  6. In Confirm Password, re-enter the same password.
    The application sets the password.
  7. To share the report with the recipients, click Send.
    The sender and the added recipients receive the report in PDF over an email message. If the report fails to generate, the sender and the recipients receive an email message indicating the same. You can retry the steps for sharing the report. If you are unable to send the report, contact Rubrik Support.
