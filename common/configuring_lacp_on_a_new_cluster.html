Configuring LACP on a new cluster

Configure a new node for LACP using the admin CLI.

Procedure

  1. Open an SSH session on the node, or access the console using the IPMI or crash cart.
  2. Type the following command. 
    network bond_mode get
    The set bond mode prompt appears.
  3. Type this command to configure the bond for LACP. 
    network bond_mode set
  4. Type yes.
    Alternatively, to keep the current mode, type no.
  5. Type LACP.
    The node sets the bond mode to LACP.
  6. Optional: Repeat these steps on other nodes in the Rubrik cluster.
    This task configures one node to use LACP. Repeat the task on each node that should use LACP.

Results

Rubrik CDM configures the nodes for LACP.

What to do next

Perform the bootstrap procedure to form the cluster. The bootstrap procedure is described in the Rubrik CDM Install and Upgrade Guide.
