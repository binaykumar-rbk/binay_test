Rewinding AI agent errors from the Explorer page

Locate AI agent errors on the Explorer page and rewind the transaction to a point in time before the errors occurred.

Procedure

  1. Log in to RSC.
  2. Click and select Agent Operations.
    The Agent Operations page appears.
  3. Click Explorer.
    The Explorer page appears, displaying a roadmap that outlines risky AI agent actions in reverse order, from their execution to their root cause. Selecting any of these actions displays the previous action.
  4. Optional: Click View by and select Agents or Applications to displays a roadmap for risky actions performed by agents or applications.
  5. Click the actions until you reach, and then click, the root cause action.
    Agent Operations opens the Rewind tray.

    The tray includes the Action Summary, which includes details about the action taken, and a violation list, which includes information about why this action violates a policy.

  6. Optional: Click Open and change the status of the AI agent error.
    You can chose from these statuses:
    • Remediated
    • Dismissed
  7. Click Remediate and then Rewind Action.
    The rewind tray moves to the Configure Rewind Details page.
  8. Optional: Change the object path or the rewind time.
    Clicking Select Object or Select Rewind Time allows you to change these parameters.
  9. Click Rewind.
    Agent Operations reverts this AI agent action to the point in time specified and returns to the Action History page. The AI agent action being rewound includes an informational message indicating the rewind status.

Results

You have located the AI agent error on the Explorer page and rewound the transaction to a point in time before the errors occurred.
