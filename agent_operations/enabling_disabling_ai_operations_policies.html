Enabling or disabling Agent Operations policies

Enable or disable policies to change which actions Agent Operations uses for governance.

Procedure

  1. Log in to RSC.
  2. Click and select Agent Operations.
    The Agent Operations page appears.
  3. Click Policies.
    Agent Operations displays the Policies page, which contains a list of the policies that govern Agent Operations.
  4. Click and select Enable Policy or Disable Policy.
  5. Click Enable or Disable.

Results

You have successfully enabled or disabled Agent Operations policies.
