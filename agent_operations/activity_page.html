Activity page

The Activity page lists AI agent actions identified by Agent Operationsas having erroneously caused changes to data or configuration.

Filters for the Activity page

The following table describes the filters for the Activity Log page.
Filter Description
Time Range Time range of the AI agent actions, filtered by time range. You can select the Custom option to specify a precise range.
Agent Risk Level of risk an AI agent's action poses to your data or configuration.
Agent Name of the AI agent that performed the reported action.
Application Application running the AI agent that performed the reported action.

Alerts table

You can select an alert to display the Remediate tray.

This table describes the columns in the Alerts table.
Column Description
Alert Alert the violation triggered.
Policies Violated Policy that was violated by the AI agent.
Agent Name of the AI agent that performed the reported action.
Status Status of the violation. it can be any one of the following:
  • Open: The violation is still active in the workload data.
  • In Progress: The violation is transitioning from Open to Resolved.
  • Resolved: Agent Operations has rolled back the violation.
Application Application issuing the action that resulted in the violation.
Tool Invoked Name of the tool that invoked the violation.
Detected on Date and time Agent Operations detected the violation.

Activity Log table

This table describes the columns in the Activity Log table.
Column Description
Action Action performed by the AI agent.
Agent Name of the AI agent that performed the reported action.
Agent Risk Level of risk associated with the agent.
Application Application running the AI agent that performed the reported action.
Identity User account that started the agent that performed the reported action.
Tool Invoked Tool the AI agent used to perform the reported action.
Time Time the AI agent performed the action.
