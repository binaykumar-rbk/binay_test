Marking tools as authorized or unauthorized

Mark tools that are available to Agent Operations as authorized or unauthorized.

About this task

You can use Agent Operations to explicitly mark specific tools an authorized or as unauthorized. Authorized tools are approved for the tasks performed by the AI agents. Unauthorized tools are not approved by the IT organization and may perform high-risk behaviors.

Procedure

  1. Log in to RSC.
  2. Click and select Agent Operations.
    The Agent Operations page appears.
  3. Click Inventory > Tools.
    Agent Operations displays the Tools Inventory page with a list of the tools available.
  4. Click for the tool to mark as authorized or unauthorized.
  5. Select the option for the tool.
    OptionDescription
    Mark as Authorized Select this option to indicate the tool is approved to perform tasks for AI agents.
    Mark as Unauthorized Select this option to indicate the tool is not approved to perform tasks for AI agents. Agent Operations will subsequently mark any tasks performed by this tool as policy violations.
  6. Confirm the selection.

Results

You have marked tools that are available to Agent Operations as authorized or unauthorized.
